Описание проекта "Симуляция жизни бактерий"



Структура:
проект содержит три директории: include, python, src:

	директория include содержит заголовочные файлы классов проекта;
	
	директория src cодержит файлы типа *.cpp, которые являются реализацией функций из соответствующих заголовочных файлов или же это файл привязок между C++ -> Python (binding1.cpp);
	
	директория python содержит общую библиотеку (particle_sim.pyd), необходимы для реализации поиска на Prolog файлы (particle_rules.pl, current_particles.pl), и файл с визуализацией симуляции (test_final.py).

файл func.hs вне описанных директорий реализует структуру данных, используемую в проекте на Haskell.


Этап 7 - связывание Prolog и Python
(участники: test_final.py, particle_rules.pl)
